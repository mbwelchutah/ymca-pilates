## PART 3 — Add client-side countdown

In the existing <script>:

1. For each row:

   * read data-open-time
   * compute time remaining

2. Create helper:

```js
function formatCountdown(ms) {
  if (ms <= 0) return "Open";

  const totalSeconds = Math.floor(ms / 1000);
  const h = Math.floor(totalSeconds / 3600);
  const m = Math.floor((totalSeconds % 3600) / 60);
  const s = totalSeconds % 60;

  return `${h}:${m.toString().padStart(2,'0')}:${s.toString().padStart(2,'0')}`;
}
```

3. Update all rows every second:

```js
setInterval(() => {
  const now = Date.now();

  document.querySelectorAll("[data-open-time]").forEach(row => {
    const openTime = new Date(row.dataset.openTime).getTime();
    const diff = openTime - now;

    const cell = row.querySelector(".countdown-cell");
    if (cell) {
      cell.textContent = formatCountdown(diff);
    }
  });
}, 1000);
```
