STAGE 9A — BACKGROUND PREFLIGHT ENGINE
================================

Goal:
Create an automatic background readiness loop.

Tasks:
1. Add a background preflight runner for active jobs
2. Run at a safe interval, such as every 2–5 minutes
3. For each active job, perform safe readiness checks:
   - verify session
   - verify schedule access
   - verify class discovery
   - verify modal reachability if safe
   - evaluate action availability if safe
4. Skip the loop if a real booking attempt is currently running
5. Avoid browser/session lock conflicts
6. Store latest result in shared state
7. Add structured logs with prefix:
