## PART 3 — Compute next job

In the countdown loop:

1. Find next job (smallest positive diff), excluding:

   * booked jobs
   * inactive jobs

2. Store:

```js
let nextJob = null;
let minDiff = Infinity;
```

---

## PART 4 — Update banner every second

```js
const banner = document.getElementById("next-job-banner");

if (!nextJob) {
  banner.classList.add("hidden");
} else {
  banner.classList.remove("hidden");

  const diff = minDiff;

  const formatted = formatCountdown(diff);

  banner.textContent = `⏳ Next job opens in ${formatted}`;

  banner.classList.remove("warning", "sniper");

  if (diff <= 60000 && diff > 0) {
    banner.classList.add("warning");
  }

  if (diff <= 0) {
    banner.textContent = "🔥 Booking open now";
    banner.classList.add("sniper");
  }
}
```
