## PART 4 — Update digit smoothly

```js
function updateDigit(el, newValue) {
  const inner = el.querySelector(".digit-inner");

  if (!inner || inner.textContent === newValue) return;

  // create next value
  const next = document.createElement("span");
  next.className = "digit-inner";
  next.textContent = newValue;

  el.appendChild(next);

  // trigger animation
  requestAnimationFrame(() => {
    el.classList.add("roll-up");
  });

  // cleanup after animation
  setTimeout(() => {
    el.classList.remove("roll-up");
    el.innerHTML = `<span class="digit-inner">${newValue}</span>`;
  }, 180);
}
```
